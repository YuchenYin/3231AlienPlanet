About
--------------------------------
Easily repack texture channels to more easily import 3rd-party texture resources, then pack them to use with Unity's built-in shader types that expect specific packing structures.


Installation Instructions
--------------------------------
1. Import the asset into your project.
2. Go to Tools -> Texture Channel Packer
3. Add the textures to the corresponding target channels
4. Specify the source channels and other properties
5. Set the file format
6. Click save to specify a target directory and file name.
